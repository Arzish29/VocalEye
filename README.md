👁️ VocalEye
🚀 Overview

VocalEye is an assistive technology application designed to help visually impaired individuals understand and interact with their surroundings using voice commands and AI.

⚙️ How It Works

VocalEye operates in two modes:

🌐 Online Mode

Uses Imagga API for real-time image recognition

When the user points the camera at any object, the app captures the image

The API processes it and provides accurate descriptions

The app then converts the result into voice output


📴 Offline Mode

Uses ML Kit for on-device object detection

Works without internet connection

Detects objects in front of the user and gives instant audio feedback

👉 In both modes, the user simply points the camera, and the app tells what is in front of them


🚨 Smart Emergency Features


📞 Voice-based Calling

User saves a number as an Emergency Contact

Just say "Call"

The app automatically calls the saved contact


📍 Live Location Sharing

Say "Share my live location"

The app sends the user's real-time location to the emergency contact
